# Xue-React-Next

一个 `React + Next.js`版本的 Halo 主题，不管是使用者还是开发者，都能带给你们不一样的体验。

使用了前后端分离之后的Halo主题，在某种程度上面已经脱离了Halo而存在。但是用户还是需要在管理后台上传[主题文件](https://github.com/xzzai/halo-theme-xue/tree/react-next/halo/xue.zip)，该文件我已经放在了`halo`目录。
