# Xue-React-Next

一个 `React + Next.js`版本的 Halo 主题，不管是使用者还是开发者，都能带给你们不一样的体验。

[DEMO](https://baozi.fun)

## Usage
### 1. First Install
```shell script
npm install 
# or cnpm install 
# or yarn 
```
### 2. Then Build
```shell script
npm run build
```

### 3. And development
```shell script
npm run dev
or yarn dev
```


### 4. Last Open Browser
Visit http://localhost:3000
